상세 페이지 헤드 스타일 업데이트 
현재 명함 컴포넌트가 분리되어 있다. 
현재 헤드 컴포넌트 명함은 제거하고 명함 도메인의 명함 컴포넌트를 적용하여 명함 중복을 제거한다.
그리고 명함은 사진을 업로드하지 않으면 기존 명함 템플릿을 적용하고 명함 사진 업로드를 적용할 수 있게하여 기본과 회사별 명함 이미지를 적용할 수 있게 한다. 
크기는 직원목록에 사이사이즈로 하여 충분한 정보판별이될 수 있게 한다.
업함 이미지 업로드시 개인첨부서류과 별도로 관리한다. 
개인 사진도 별도로 관리하며 따로 개인첨부서류로 관리할 필요가 없다.
