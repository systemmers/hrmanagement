법인설정 > 조직관리

- 조직추가 버튼도 아이콘 버튼으로 통일 
- 인원 수는 코드명 옆이아니라 조직명 옆에 표시
- 조직유형 설정은의 구조는 조직은 단위는 여러개일 수 있으며 많이 사용하는 단위 디폴트 값을 7개를 세팅한것이다.
이 세팅값은 사용자 회사조건에 맞게 수정할 수 있다. 아이콘도 선택할 수 있으며 단계의 순서도 수정할 수 있다. 
조직의 단위 입력시 한글명, 영어명으로 입력할 수 있음. 활성화는 의미가 없으며로 삭제한다.
7개 단계는 조직트리의 뎁스를 언급한 것이고 뎁스를 떠나 뎁스가 낮아도 다양한 부서가 많을 수 있기 때문이고 같은 뎁스에 다양한 단위 및 유형의 조직이 존재할 수 있다. 
- 조직장의 내선번호가 아니라 조직의 대표 내선번호, 조직의 이메일로 변경한다.
- 추가된 조직정보에 출력되게하며, 조직의 인원도 출력되게 하여라.
