## 1. Executive Summary

### 1.1 프로젝트 정의

**WorkBridge**는 개인의 검증된 경력 정보(사실)와 차별화된 포트폴리오를 분리하여 관리하며, 기업의 근로기준법 자동 준수와 채용 효율화를 동시에 실현하는 혁신적인 HR 플랫폼입니다.

### 1.2 핵심 가치 제안

```
개인 (근로자)
├─ 100% 검증된 경력 관리 (허위 이력 불가능)
├─ 회사별 맞춤 포트폴리오 제출
├─ 평생 경력 데이터 소유권 보장
└─ AI 기반 직무 매칭

기업 (고용주)
├─ 이력서 검증 비용 Zero
├─ 채용 기간 2개월 → 2주
├─ 근로기준법 자동 준수 (과태료 위험 제거)
├─ 정부 지원금 신청 자동화
└─ 인사 업무 시간 80% 절감

정부
├─ 허위 이력서 100% → 0%
├─ 근로감독 효율 100배 향상
├─ 실시간 고용 통계 확보
└─ 지원금 부정수급 차단
```

### 1.3 시장 기회

- **TAM:** 18조 원 (국내 전체 HR 시장)
- **SAM:** 9,600억 원 (직원 5인 이상 기업)
- **Year 5 목표:** 600억 원 매출, 50,000개 기업 고객

### 1.4 핵심 경쟁력

1. **세계 최초 사실-포트폴리오 이원화 시스템**
   - 사실 정보: 100% 검증 (학력, 경력, 자격증)
   - 포트폴리오: 자유로운 차별화 (프로젝트, 성과, 스토리)

2. **법적 컴플라이언스 자동화**
   - 근로기준법 위반 원천 차단
   - 법령 변경 실시간 추적 및 자동 대응

3. **정부 연동 가능성**
   - 4대보험 API 연동
   - 지원금 신청 자동화
   - 국가 고용 인프라로 진화 가능

## 2. 프로젝트 개요

### 2.1 비전

**"대한민국 모든 근로자의 디지털 경력 여권"**

2030년까지 전 국민 근로 이력을 투명하게 관리하고, 정부-기업-개인이 신뢰하는 국가 고용 인프라가 되는 것

### 2.2 미션

1. 근로자의 권익을 디지털로 보호한다
2. 허위 이력을 기술적으로 차단한다
3. 중소기업의 법적 리스크를 제거한다
4. 정부의 행정 효율을 극대화한다

### 2.3 핵심 가치

- **신뢰 (Trust):** 100% 검증된 사실 정보
- **차별화 (Differentiation):** 개인의 스토리텔링 포트폴리오
- **효율 (Efficiency):** 자동화된 HR 프로세스
- **컴플라이언스 (Compliance):** 법 위반 불가능한 구조

## 3. 배경 및 필요성

### 3.1 현실의 문제점

#### 3.1.1 기업의 문제

**통계:**
- 중소기업 80-90%가 근로기준법 미준수
- 근로감독 위반율: 68%
- 평균 과태료: 347만 원
- 서류 급조 비용: 평균 500만 원/건

**실제 사례:**
```
A제조사 (직원 30명)
상황:
- 근로계약서 미작성: 18명
- 임금대장 없음
- 연차관리 구두로만 진행
- 수습평가 기록 없음

근로감독 통보 후:
- 2주간 밤샘 근무로 서류 급조
- 날짜 "추정"으로 작성
- Excel 수정일 감추기 시도

결과:
- 과태료 850만 원
- 시정명령 3건
- 대표 형사고발 위기
```

#### 3.1.2 개인(근로자)의 문제

**이력서 허위 현황:**
- 학력 위조: 8-12%
- 경력 과장: 20-30%
- 자격증 위조: 5-8%
- 프로젝트 경험 부풀리기: 40%+

**문제 상황:**
```
B씨 (IT 개발자, 경력 6년)

이직 시 매번:
- 재직증명서 발급 요청 (전 직장 3곳)
- 경력기술서 새로 작성
- 졸업증명서 발급 (우편 2주)
- 2번째 회사 폐업 → 경력 증명 불가

새 회사 입사 서류 준비: 총 3주 소요
```

**진짜 실력 있는 사람도 피해:**
```
실제 박사 학위 소지자:
- 면접관: "학위 진짜예요? 요즘 위조 많던데..."
- 추가 검증 요구
- 불쾌한 경험

결과: 검증된 시스템의 부재로 
     모두가 의심받는 구조
```

#### 3.1.3 정부의 문제

**근로감독 비효율:**
```
1건 처리 프로세스:
1. 서류 요청 → 기업에 14일 기한 부여
2. 제출 서류 검토 → 5일
3. 현장 방문 조사 → 2일
4. 직원 면담 → 3일
5. 보고서 작성 → 5일

총: 29일 소요
조사관 1명 연간 처리: 약 12건

문제: 
- 전국 360만 개 기업
- 조사관 부족
- 실질적 감독 불가능
```

**제출 서류 신뢰성 문제:**
```
노동부 조사관 증언:
"제출된 서류 대부분이 감독 통보 후 
 급조한 것임을 알지만 증명할 방법이 없습니다.
 
 Excel 파일은 언제든 수정 가능하고
 PDF로 변환하면 생성일도 숨길 수 있어요.
 
 결국 직원 면담으로 확인하는데
 직원들도 회사 눈치 보느라 
 사실대로 말 안 합니다."

→ 조사 신뢰성: 30% 이하
```

**정부 지원금 허위 수급:**
```
R&D 지원금 심사관:
"제출된 경력 서류의 40%가 의심스럽습니다."

흔한 허위 사례:
- 박사 학위 없는데 있다고 기재
- 재직 기간 부풀리기 (3년 → 5년)
- 프로젝트 경험 조작
- 타사 직원을 자사 직원으로 위장

문제: 
- 일일이 검증 불가능
- 허위 수급 추정 연간 수천억 원
```

**검증 프로세스:**
```
Step 1: 사용자가 학교명 입력
  ↓
Step 2: 학교 선택 (드롭다운, 인증된 학교만)
  ↓
Step 3: 학교 이메일로 인증 코드 발송
  예: hong@snu.ac.kr
  ↓
Step 4: 인증 완료 후 학교 API 조회
  (또는 졸업증명서 PDF 업로드 → OCR)
  ↓
Step 5: 학적 정보 자동 로드
  - 학번, 전공, 학위
  - 입학일, 졸업일
  - 학점 (선택적)
  ↓
Step 6: 블록체인 타임스탬프 기록
  - 등록 시각
  - 데이터 해시
  - 변경 이력
```

**검증 방법 (3가지):**

1. **회사가 WorkBridge 사용 중 (최선)**

## 15. 결론

### 15.1 핵심 요약

WorkBridge는 **사실과 포트폴리오의 이원화**라는 혁신적 개념으로:

1. **개인**에게
   - 100% 검증된 평생 경력 관리
   - 회사별 맞춤 포트폴리오 제출
   - 허위 없는 정직한 경쟁 환경

2. **기업**에게
   - 이력서 검증 비용 Zero
   - 채용 기간 대폭 단축
   - 법적 컴플라이언스 자동 준수
   - 정부 지원금 신청 자동화

3. **정부**에게
   - 허위 서류 100% 차단
   - 행정 효율 극대화
   - 실시간 고용 데이터 확보

### 15.2 차별화 포인트

```
경쟁사:
"편리한 HR 관리 도구"
"이력서 플랫폼"

WorkBridge:
"검증된 사실 + 자유로운 스토리"
"법을 자동으로 지켜주는 필수 인프라"

→ 선택이 아닌 필수
→ 허위 불가능한 신뢰
→ 개인의 진짜 차별화
```
