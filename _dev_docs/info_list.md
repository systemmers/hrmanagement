# 인사기록카드 페이지별 항목 리스트



## 1. 기본정보 페이지

### 1.1 소속정보
- 회사
- 소속부서(부서/팀)
- 직급/직책
- 입사일자/퇴사일자
- 고용형태(정규직, 계약직, 무기/기간, 인턴 등)
- 근무지
- 사원번호 (자동 부여, 사용한 번호는 사용금지)

### 1.2 기본정보 : 소속과 관련없는 순수정보

- 사진
- 성명(한글/영문)
- 영어이름
- 주민등록번호/외국인등록번호
- 생년월일 (주민등록번호로 자동 인식, 수정가능)
- 성별 (주민등록번호로 자동 인식)
- 연락처(휴대폰/비상연락처)
- 비상연락처(연락처, 구분)
- 이메일(회사, 본인)
- 주소(주민등록상/실거주지)

### 1.3 병역정보
- 병역상태 (군필/면제/미필/해당없음)
- 군별 (육군/해군/공군/해병대)
- 계급
- 복무기간
- 전역일자
- 보훈대상여부
- 보훈유형

### 1.4 장애정보
- 장애여부
- 장애등급
- 장애유형

## 2. 계약정보 페이지

### 2.1 현재 계약정보
- 근로계약 유형
- 계약기간(시작일/종료일)
- 직무 및 담당업무
- 급여형태 및 금액
- 근로시간(시업/종업/주간근로시간)
- 휴게시간
- 계약 갱신 여부
- 계약서 파일 첨부

### 2.2 근로계약정보 이력 (테이블 형식)
**테이블 정보:**
- 근로계약 유형
- 체결일
- 계약기간(시작/종료)
- 고용형태(정규직/계약직/인턴 등)
- 계약상태

**클릭시 세부정보:**
- 직무 및 담당업무
- 근로시간(시업/종업/주간근로시간)
- 휴게시간
- 시용기간, 수습기간 (1년 미만내 적용)
- 수습평가
- 평가서 연동
- 근로계약서 스캔본 연동

### 2.3 연봉계약정보 이력 (테이블 형식)
**테이블 정보:**
- 계약년도
- 체결일
- 계약기간(시작/종료)
- 급여형태(월급/시급/연봉)
- 계약상태

**클릭시 세부정보:**
- 급여 세부내역 및 구조
- 지급계좌
- 연봉계약서 스캔본 연동

## 3. 인사기록 페이지

### 3.1 호봉/승급/승진 이력 (테이블 형식)
**테이블 정보:**
- 구분(승진/승급/호봉)
- 이전 직급/호봉
- 변경 직급/호봉
- 발령일자
- 상태

**클릭시 세부정보:**
- 사유
- 평가서 연동

### 3.2 부서이동/발령 이력 (테이블 형식)
**테이블 정보:**
- 구분(부서이동/발령/파견)
- 이전 부서
- 변경 부서
- 발령일자
- 상태

**클릭시 세부정보:**
- 이전 직책
- 변경 직책
- 이전 근무지
- 변경 근무지
- 사유

### 3.3 평가 이력 (테이블 형식)
**테이블 정보:**
- 평가기간(3개월/6개월/1년)
- 평가일자
- 평가등급(S/A/B/C/D)
- 후속조치

**클릭시 세부정보:**
- 평가점수
- 후속조치 상세(승진/승급/유지/경고 등)

### 3.4 급여이력/변동내역 (테이블 형식)
**테이블 정보:**
- 적용일자
- 급여형태(월급/시급/연봉)
- 이전 급여액
- 변경 급여액
- 변동률
- 변경사유

**클릭시 세부정보:**
- 변경사유 상세(승진/승급/계약갱신/기타)
- 승진이력 연동
- 평가이력 연동
- 연봉계약서 연동
- 비고
