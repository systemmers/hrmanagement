## 🔑 핵심 특징

### 자동 계산 필드
- 생년월일 → 자동 입력
- 경력 기간 → 자동 계산
- 프로젝트 참여 기간 → 자동 계산
- 총 경력 → 자동 계산

### 데이터 수집 범위
- **기본정보**: 13개 주요 필드
- **학력**: 6단계 (고등학교 ~ 박사)
- **직장 경력**: 최대 10개
- **자격증**: 최대 5개
- **언어능력**: 최대 3개
- **프로젝트**: 최대 8개 (Page 1) + 5개 (Page 2)
- **연차 관리**: 5년치

### 승인 프로세스
- 작성자 → 팀 → 승인 → 재무/기획
